# Install

Release packaging is configured, but the first release tag has not been cut yet. Build from source until `v0.1.0` exists:

```sh
git clone https://github.com/tomnagengast/compactor
cd compactor
go build ./cmd/compactor
./compactor --help
```

For development checks:

```sh
dev/agent/check-fast
```

After the first release is published, install with Homebrew:

```sh
brew tap tomnagengast/tap
brew install --cask tomnagengast/tap/compactor-cli
compactor --version
```

Release assets are generated by GoReleaser and include cross-platform archives plus `checksums.txt`.
